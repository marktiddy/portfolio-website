My Work

Youth Work Resource Website
A website I run using Wordpress which has been running for over 10 years and has a high hit count each month. The website provides resources for church-based youth workers.

100 Questions - Youth (iOS)
This Youth Work app was designed by me using XCode and by programming in Swift. The app is programmed to display 1 of a bank of 100 questions to the user and is programmed to not repeat any questions during a session.

Catch 'Em Game (iOS App)
An app designed in Xcode using SpriteKit which challenges users to catch and match coloured balls at varying levels of difficulty. The app serves adMob apps in order to generate a small income.

USA Tip Calculator (iOS App)
A simple yet effective tool to help users calculate the amount they wish to tip based on their bill.

This or That (iOS App)
An app created using Xcode which presents users with 'either or' choices to make and logs their choice to a Firebase database and shows them how others have voted.
